[파일입출력]
디스크에 저장하는 방식
영구저장을 하기위해 사용한다 (해당 디스크가 망가지기전까지 사용가능)
File path = new File("(위치)");

stream - 연결통로
FileOutputStream - 파일에 내용을 입력하는 객체

.write - byte 또는 int로 받아야하는 입력메소드이다 
하지만 문자열을 넣고싶은 경우엔 .getBytes() 메소드를 write();에 넣어준다

.exists() - 해당하는 파일에 접근할수있는지 반환해주는 메소드.

FileInputStream - 파일의 내용을 출력하는 객체
.read(); 로 파일의 내용을 가져오는데 int형의 자료로만 가져오므로 반복문을 사용해서 가져와야한다
그리고 (char)로 형변환해주어야한다

BufferedOutputStream - 보조스트림으로서 파일로 직접 접근하지 못한다
그래서 버퍼에 미리 저장해놓고 .flush();로 최종입력시킨다

Stream이라고 이름붙은 객체들은 말그대로 통로이기때문에 
생성한역순으로 .close();메소듣를 이용하여 닫아주어야한다

직렬화 - 객체가 byte형식으로 변환되는 것 반대는 역직렬화라고한다
