<자바스크립트>
자바와 비슷하지만 자바의 문법을 사용할수는 없다
html은 틀 css는 디자인 자바스크립트는 기능의 역할을 맡는다

자바와 다르게 자료형이 var하나뿐이다
문장의 끝에 세미콜론(;)은 생략 가능하다

document - 현재 페이지를 의미한다

console.log - 콘솔에 출력해준다

const - 자바에서 final과 같은의미이다

prompt - 입력을 받을수있다 (스크립트안에서 만든 변수는 스크립트에서만 사용할수있다)

<연산자>
기본적인 언어들의 연산자는 동일하다

== (타입을 신경쓰지않고 값이 같은지만 확인한다)
=== (타입과 값이 모두 같은지 확인한다)

if문 for문 모두사용가능하다

function - 함수를 선언한다
var 선언을 해주면 그선언한 지역에서만 사용가능하고
선언을 해주지않으면 전역변수처럼 사용할수있다 (호출 순서를 지켜주어야한다)

보통 전역변수는 var 지역변수는 let을 많이 사용한다

자바스크립트의 함수도 return값이 존재한다
하지만 자바와다르게 반환타입이 없기때문에 var로 처리가능하다 

submit - form안에서만 사용가능하며 form속성의 action으로 이동시킨다
	form안의 태그들중 name을 갖고있는 태그들의 입력된 정보도 함께 넘겨준다

button - 버튼은 단순히 버튼의 역할을 하며 submit같은 기능을 넣으려면 onclick로 위치를 지정해주어야한다
onclick="location.href='(위치)'"
