[switch문] - switch문 말 그대로 조건에 따라 선택하는 것으로
	   아래와 같은 형태이다
	  switch(조건){
	     case 1:
		종속문장;
	     break;
	     default:
		종속문장;
	  }   
	   조건의값과 case의 값이 일치하는 경우에만 종속문장을 실행하며
	   break; 를 만나서 switch문을 벗어난다
	   default는 조건의 값이 case와 일치하는것이 없을때 실행하는 것이다
	   생략가능하다
	   마지막 case와 default가 없을경우엔 break;를 생략 할 수 있다

[while문] - 반복문으로서 조건식이 참일 경우 반복적으로 종속문장들을 실행한다
	 조건식이 거짓이거나 break;문을 실행할때까지 무한하게 종속문장들을 반복한다
 	 while(조건식){
	     종속문장;
	  }
	 break; - 반복문을 종료시킨다.
	 continue; - while문의 가장 처음으로 이동시킨다.

[변수 초기화] - 선언한 변수에 아무겂을 넣지않을경우 조건문에서 오류가 발생할수 있으므로 변수를 초기화해서
	       오류가 생기는것을 방지한다.
	       int = 0;
	       char = 0;
	       double = 0;
	       String = "";
