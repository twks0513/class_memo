final - 변수 선언시에 자료형 앞에 붙이면 해당 변수는 값이 절대 바뀌지 않는다.
        보편적으로 final로 변수를 생성시엔 변수명을 대문자로 한다.

[입력]
Scanner - 사용자로부터 입력받는 객체로서 java.util안의 Scanner를 import해야한다.

보통 Scanner scan = new Scanner(System.in); 로 많이 쓴다.

scan.next(); - 공백전까지 입력받는 모든 값을 받는다

scan.nextInt(); - 정수만을 입력받는다

scan.nextLine(); - 엔터전까지 입력받은 문자열을 입력받는다

[기본연산자]
+ 덧셈
- 뺄셈
* 곱셈
/ 몫 (정수/정수)
(만일 두 수중 하나라도 실수형이 있다면 나눈 값이 나온다)
% 나머지 (정수%정수)

[복합연산자]
a+=b -> a = a+b와 같은 뜻이다
a-=b -> a = a - b
a*=b -> a = a * b
a/=b -> a = a / b 
a%=b -> a = a % b

[관계연산자] - 결과적으로는 참과 거짓만 표현한다 
	      또한 이항연산자로서 연산자사이 피연산자가 2개만 올 수 있다
a > b 
a < b
a >= b
a <= b
a == b 두수가 같다
a != b 두수가 다르다

[논리연산자] 여러 조건에 대한 참 또는 거짓결과
&& - 그리고(and) 모두가 참일경우에 true 반환
|| - 또는(or) 전부중 하나라도 참일경우 ture반환
! - 부정(not) 결과를 반전시킨다 

[증감연산자] 결과적으로 자기자신을 1씩 증가 또는 감소시킨다

a = 10; 일때
++a 또는 a++ -> a + 1 = 11

--a 또는 a-- -> a - 1 = 9

증감연산자가 앞에 있을 경우엔 다른 연산자 보다 먼저 자기자신을 증감시킨다 
증감연산자가 뒤에 있을 경우엔 모든 연산이 끝난후에 자기자신을 증감시킨다

[삼항연산자] 
변수 = 식 ?참(값) : 거짓(값);  의 형태이고 
		            해당하는 식의 참,거짓여부를 판별하고 
			참일 경우엔 참에 해당하는 값을 변수에 저장하고
			거짓일 경우엔 거짓에 해당하는 값을 변수에 저장한다

[제어문(if문, switch문, for문, while문, do while문)] 프로그램의 흐름을 제어한다

[if문]      조건식이 참이면 종속문장을 실행한다 
	조건이 여러개일경우 else if 를 사용하고 조건식 외의 모든 조건은 else를 사용한다
if(조건식){
   (종속문장)     - 다음과 같은 형태를 띄고있다 
}
